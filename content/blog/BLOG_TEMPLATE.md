# Blog Post Template

Use this template to create new blog posts. Copy this file and rename it with your desired slug.

## File Naming Convention

- Use kebab-case for file names (e.g., `my-awesome-blog-post.md`)
- The filename becomes the URL slug (e.g., `/blog/my-awesome-blog-post`)

## Frontmatter Template

```yaml
---
title: "Your Blog Post Title Here"
excerpt: "A brief description of what this blog post is about. Keep it engaging and under 160 characters for SEO."
date: "YYYY-MM-DD"
readTime: "X min read"
category: "Category Name"
year: "YYYY"
upvotes: 0
tags: ["Tag1", "Tag2", "Tag3", "Tag4"]
---
```

## Content Guidelines

### Writing Tips:

1. **Be Authentic**: Write in your own voice and share personal experiences
2. **Be Helpful**: Provide value and actionable advice
3. **Be Engaging**: Use storytelling and examples
4. **Be Clear**: Use simple language and clear structure
5. **Be SEO-Friendly**: Include relevant keywords naturally

### Technical Requirements:

- Use proper heading hierarchy (H1, H2, H3)
- Add section IDs for table of contents: `{#section-id}`
- Include relevant tags (4-6 tags recommended)
- Set appropriate read time (aim for 5-10 minutes)
- Use proper date format (YYYY-MM-DD)

### Categories Available:

- Personal Development
- Technology
- Career
- Life Lessons
- Growth
- Productivity
- Mental Health
- Finance
- Relationships

### Tag Examples:

- Personal Development: "Growth", "Self-Improvement", "Mindset", "Goals"
- Technology: "Programming", "AI", "Web Development", "Tools"
- Career: "Career Change", "Networking", "Skills", "Leadership"
- Life Lessons: "Experience", "Wisdom", "Challenges", "Success"

## Example Usage

1. Copy this template file
2. Rename it to your desired slug (e.g., `my-new-blog-post.md`)
3. Update the frontmatter with your content
4. Replace the template content with your actual blog post
5. Add proper section IDs for navigation
6. Test the blog post by visiting `/blog/your-slug`

## File Location

Place your new blog post in: `/content/blog/your-blog-post.md`

The blog system will automatically:

- Parse the frontmatter
- Generate the blog post page
- Add it to the blog listing
- Enable upvoting functionality
- Create table of contents navigation
